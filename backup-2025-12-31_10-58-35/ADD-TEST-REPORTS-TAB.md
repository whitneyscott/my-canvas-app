# Adding Test Reports Tab - Complete Instructions

## Step 1: Add to Tabs Array

In `views/index.ejs`, find the `tabs` array (around line 1098) and add:

```javascript
const tabs = [
    // ... existing tabs ...
    { 
        name: 'test-reports', 
        displayName: 'Test Reports', 
        isActive: false
    }
];
```

## Step 2: Add Tab Button

In the tabs container (around line 1086), add the button:

```html
<div class="tabs">
    <!-- ... existing buttons ... -->
    <button class="tab-button" onclick="switchTab('test-reports')">Test Reports</button>
</div>
```

## Step 3: Add Tab Content

The tab content will be rendered by the partial. You can either:

**Option A:** Create a partial file `views/partials/test-reports-tab.ejs`:

```html
<div class="tab-content" id="test-reports-content" style="display: none;">
    <!-- Content populated by displayTestReports() -->
</div>
```

**Option B:** Add directly in the main template after the tabs loop:

```html
<div class="tab-content" id="test-reports-content" style="display: none;">
    <!-- Content populated by displayTestReports() -->
</div>
```

## Step 4: Update Tab Switching Logic

In your `switchTab()` function, add:

```javascript
function switchTab(tabName) {
    // Hide all tab contents
    document.querySelectorAll('.tab-content').forEach(content => {
        content.style.display = 'none';
    });
    
    // Remove active class from all buttons
    document.querySelectorAll('.tab-button').forEach(btn => {
        btn.classList.remove('active');
    });
    
    // Show selected tab
    const tabContent = document.getElementById(`${tabName}-content`);
    const tabButton = document.querySelector(`[onclick="switchTab('${tabName}')"]`);
    
    if (tabContent) {
        tabContent.style.display = 'block';
    }
    if (tabButton) {
        tabButton.classList.add('active');
    }
    
    // Special handling for test reports
    if (tabName === 'test-reports') {
        if (typeof displayTestReports === 'function') {
            displayTestReports();
        }
    }
    
    // ... rest of your tab switching logic ...
}
```

## Step 5: Include Scripts

Make sure both scripts are included (already done in current index.ejs):

```html
<script src="/test-system.js"></script>
<script src="/test-reports.js"></script>
```

## Step 6: Initial Load (Optional)

If you want to show reports when the page loads and test-reports is the active tab:

```javascript
document.addEventListener('DOMContentLoaded', () => {
    // ... existing code ...
    
    // Check if test-reports tab should be shown
    const urlParams = new URLSearchParams(window.location.search);
    const tab = urlParams.get('tab');
    if (tab === 'test-reports') {
        switchTab('test-reports');
    }
});
```
